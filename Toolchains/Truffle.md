## usage
step|expl.
----|----
```install```|```npm install -g truffle```
```new.project```|```truffle init```
```new.contract```|```truffle create contract <contractName>```
```test```|```truffle test contract <contractName>```
```compile```|```truffle compile```
```migrate```|```truffle migrate```
